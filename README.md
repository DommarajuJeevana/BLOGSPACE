# Blog Platform
A full-stack blogging platform where users can create, publish, edit, and manage blog posts.

## Features
- User authentication
- Create blog posts
- Edit posts
- Delete posts
- Read blogs
- Search blogs
- Category-based filtering
- Responsive design

## Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Clone Repository

git clone <repository-url>

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
npm run server

## Usage
- Register/Login
- Publish blogs
- Read blogs
- Search articles
- Manage your posts
